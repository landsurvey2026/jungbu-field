# 🔧 관리자 기능 적용을 위한 추가 설정

새 파일에 **관리자 패널** 기능이 추가되어, Firestore 보안 규칙을 **딱 한 번 더** 업데이트해야 합니다. 약 2분이면 끝납니다.

---

## 보안 규칙 업데이트

1. https://console.firebase.google.com 접속 → `arisu-central` 프로젝트 클릭
2. 좌측 **"Firestore Database"** 클릭
3. 상단 탭에서 **"규칙"** 클릭
4. 편집기의 **기존 코드 모두 삭제**
5. 아래 코드를 **그대로** 붙여넣기:

```
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {

    // 설정 (비밀번호, 공유 주소)
    match /config/{doc} {
      allow read, write: if request.auth != null;
    }

    // 관리자 PC 등록 정보
    match /adminDevices/{deviceId} {
      allow read, write: if request.auth != null;
    }

    // 공사지시
    match /orders/{orderId} {
      allow read, write: if request.auth != null;

      match /chat/{messageId} {
        allow read, write: if request.auth != null;
      }
    }
  }
}
```

6. 오른쪽 위 **"게시"** 클릭
7. 게시 완료 후 약 1분 기다리기

---

## 첫 관리자 로그인

새 파일에는 **기본 관리자 비밀번호** 가 들어있어요. 처음에는 이 값으로 로그인하시면 됩니다.

- **기본 관리자 비밀번호**: `!2026`

(HTML 파일에서 `DEFAULT_ADMIN_PASSWORD` 검색하면 코드에서 직접 확인/변경 가능합니다)

### 첫 관리자 로그인 절차

1. 본인 PC 에서 프로그램 접속
2. 비밀번호 칸에 `!i20262026` 입력
3. 이름 입력 → 접속
4. → 자동으로 본인 PC가 **첫 번째 관리자 PC** 로 등록됨
5. 우측 상단 **"⚙️ 관리자"** 버튼 클릭
6. 관리자 패널 진입
7. **첫 작업으로 비밀번호 두 개 다 변경 권장**:
   - 사용자 비밀번호 → 직원들과 공유할 새 비밀번호
   - 관리자 비밀번호 → 본인만 알 새 비밀번호 (잊지 마세요!)

### 두 번째 관리자 PC 등록

부관리자 1명을 더 두고 싶다면:
1. 그 사람 PC 에서 위 절차와 같이 `!2026` (변경했으면 새 비밀번호) 으로 로그인
2. 자동으로 두 번째 관리자 PC 로 등록됨
3. 이후 세 번째 PC 는 자동 차단됨

### 관리자 PC 변경

기존 관리자 PC 를 바꾸려면:
1. 본인이 관리자 패널 → "등록된 관리자 PC" → 바꿀 PC 옆 **"제거"** 클릭
2. 새 PC 에서 관리자 비밀번호로 로그인 → 자동 등록

## 자주 묻는 질문

**Q. 모바일에서도 관리자 가능한가요?**
A. ❌ 안 됩니다. 보안상 관리자는 **PC에서만 접속** 가능합니다. 모바일에서 관리자 비밀번호 입력 시 "관리자는 PC에서만 접속 가능합니다" 메시지가 표시됩니다.

**Q. 관리자 비밀번호를 까먹었어요!**
A. HTML 파일을 열고 `DEFAULT_ADMIN_PASSWORD` 검색 → 값을 새 비밀번호로 수정 → GitHub 에 다시 업로드 → 그 비밀번호로 다시 로그인. 그러나 변경한 비밀번호는 Firestore 에 저장되어 있어 그게 우선이므로, Firebase 콘솔에서 직접 삭제도 필요합니다 (Firestore Database → config → main → adminPassword 필드 삭제).

**Q. 사용자 비밀번호와 관리자 비밀번호는 같아도 되나요?**
A. 기술적으로는 됩니다만, 같으면 어떻게 구분하나요? 시스템은 관리자 비밀번호를 먼저 체크하므로 같으면 모든 사람이 관리자가 됩니다. **반드시 다르게 설정하세요.**

**Q. 한 명이 관리자 + 사용자 둘 다 쓰고 싶어요**
A. 가능합니다. 관리자로 로그인하면 자동으로 사용자 기능 + 관리자 패널 모두 사용 가능합니다.
