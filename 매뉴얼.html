4. 공사지시 상세 화면

메인 화면에서 카드/행을 클릭하면 상세 모달이 열려요. 여기서 모든 관리 작업을 합니다.

공사지시 상세 화면 구조 (섹션별)

🎯 각 섹션 상세

1️⃣ 기본 정보

지시번호·공사명·위치·지시일·연장·소장·연락처. 우측 상단 ✏️ 편집 아이콘으로 수정.

2️⃣ 공사 진행 (5단계)

각 단계 행을 클릭하면 상태 전환 + 날짜 자동 기록. 뒤로 되돌리려면 다시 클릭.

1. 공사예정 → 2. 공사중 → 3. 공사완료 → 4. 측량완료 → 5. 로딩완료

3️⃣ 부분로딩 / 가로딩

4단계(측량완료) 이상에서만 활성화. 체크하면 카드 상단에 보라색 뱃지 표시.

4️⃣ 준공도 인수

체크박스 + 인수일. 카드에 ✓ 준공도 태그 표시.

5. ☁️ 전산팀 입력 (핵심!)

🎯 여기가 전산팀(우리)의 주요 작업 영역
현장 정보는 엑셀에서 자동으로 오지만, 실연장과 ☁️ 전산 로딩 완료는 우리가 직접 입력해야 합니다.

📏 실연장 입력

STEP 1

지시서 클릭

메인 화면에서 로딩 대상 지시서를 클릭해서 상세 화면 열기.

STEP 2

실연장 입력

노란 배경의 "☁️ 전산팀 입력 영역"에서 실연장 값 입력 (예: 42.5).

STEP 3

저장

💾 실연장 저장 버튼 클릭. Toast로 "42.5m 저장됨" 알림.

☁️ 전산 로딩 완료 체크

이 체크박스를 ON으로 두면:

  • 메인 화면 카드의 배경이 하늘색으로 바뀝니다 (한눈에 구분)
  • 📊 실적 통계에 "우리 전산팀이 완료한 것"으로 집계됩니다
  • 체크한 날짜가 자동으로 기록됨 (월별 통계 기준)
💡 실연장은 어떻게 엑셀에 나가나요?
실연장 값이 엑셀 내보내기 시 N열(편집)과 O열(로딩) 두 곳에 동일하게 기입됩니다. 편집연장과 로딩연장을 따로 관리하지 않고 하나로 통합했어요.

10. 📤 엑셀 내보내기

메인 화면 툴바의 📥 엑셀 버튼 클릭 → 자동 다운로드.

📄 파일 구조 (원본 양식 100% 준수)

엑셀 파일 구조 (원본 관리대장 형식)

📊 5~8행 통계 (자동 수식)

  • 행 5 (계): 신규 총 건수 + G~O열 전체 합계 (=SUM 6:8)
  • 행 6 (삼아 소계): =SUMIFS(AH="삼아")
  • 행 7 (삼인 소계): =SUMIFS(AH="삼인*") — 삼인 2팀 포함
  • 행 8 (올포 소계): =SUMIFS(AH="올포*") — 1팀·M·G·N 모두 포함
  • 우측 W~AA: 진행상태별 COUNTIFS (미시공/공사중/공사완료/로딩/자료인수)

📐 데이터 행 (수식 포함)

  • H (공사실적): =I{행} (계 열을 그대로 참조)
  • I (계): =J{행}+K{행}+L{행} (조사+탐사+불탐 합)
  • J/K/L/M: 실제 값 (조사/탐사/불탐/측량)
  • N/O (편집/로딩): 앱에서 입력한 실연장 값이 동일하게 들어감
💡 J·K·L만 바꿔도 자동 재계산
엑셀 열어서 조사/탐사/불탐 값을 바꾸면 I(계)와 H(공사실적)가 자동으로 재계산되어 통계도 함께 업데이트됩니다.

12. ❓ 자주 묻는 질문

Q. 로그인이 안 돼요. 비밀번호가 틀렸다고 나옵니다.

비밀번호는 대소문자를 구분하지 않지만 특수문자와 숫자는 정확해야 합니다.

  • 일반: j2026 (앞의 j는 소문자 j)
  • 관리자: !!2026 (느낌표 2개 + 2026)

여전히 안 되면 시크릿 창(Ctrl+Shift+N)에서 다시 시도해보세요. 브라우저 캐시 문제일 수 있습니다.

Q. 관리자로 로그인했는데 ⚙️ 아이콘이 안 보여요.

관리자는 PC에서만 로그인 가능합니다. 스마트폰에서는 관리자 비밀번호를 입력해도 승인되지 않아요.

또한 관리자 PC 최대 2대 제한이 있어서, 이미 다른 PC 2대가 등록된 경우 3번째 PC에서는 접속이 거부됩니다.

Q. 엑셀을 임포트했는데 제가 입력한 실연장이 사라졌나요?

아니요. v3.3부터 임포트 정책이 강화되어 실연장·로딩완료·전산로딩 체크·준공도인수 등 전산팀이 입력한 값은 절대 덮어쓰지 않습니다.

임포트되는 것은 신규 지시번호와, 기존 지시의 현장 필드(컨소담당·현장특이사항·주야·구경·추진실적 등)뿐입니다.

Q. 카드 배경이 하늘색인데 뭘 뜻하나요?

☁️ 전산 로딩 완료 체크박스가 켜져 있다는 뜻입니다. 우리 전산팀이 해당 지시서를 로딩 처리했다는 표시예요.

📊 실적 통계에서 "☁️ 전산 로딩" 누계에 포함됩니다.

Q. 엑셀에 값이 안 들어가고 ####로 나와요.

엑셀 셀 폭이 좁아서 나타나는 표시입니다. 해당 열 헤더를 더블클릭하면 자동으로 폭이 조정돼요. v3.1부터 폭을 넉넉히 설정했지만 여전히 좁으면 수동 조정 가능합니다.

Q. 부분로딩 / 가로딩이 뭐가 다른가요?

둘 다 측량완료(4단계) 상태에서 로딩 진행 중인 것을 표시합니다.

  • 부분로딩: 배관의 일부분만 로딩 완료
  • 가로딩: 임시로 로딩 처리 (실제 완료 전 임시 상태)

둘 다 카드 상단에 보라색 뱃지로 표시되고 별도 탭에서 필터링할 수 있습니다.

Q. 알림 위젯이 화면을 가려서 불편해요.

위젯을 드래그해서 원하는 위치로 이동할 수 있어요. 위치가 자동 저장되어 다음 방문 시에도 유지됩니다. − 버튼으로 최소화하거나 × 버튼으로 세션 동안 숨길 수도 있습니다.

Q. 새 지시서를 등록하려면 어떤 정보가 꼭 필요한가요?

필수 항목은 지시번호와 공사명 두 가지입니다. 나머지는 나중에 채우거나 엑셀 임포트 시 자동으로 병합돼요.

지시번호는 숫자만 입력하면 앞에 자동으로 "중부"가 붙습니다 (001 → 중부001).

Q. 채팅에 남긴 메시지는 누가 보나요?

각 공사지시별로 독립된 채팅이며, 앱에 로그인한 모든 사람이 볼 수 있습니다. 팀 내 소통용이니 현장 특이사항 · 이슈 공유 등에 활용하세요.

Q. 앱 버전 업데이트 후 새 기능이 안 보여요.

브라우저 캐시 때문입니다. 다음 순서로 해결해보세요:

  1. 주소창 뒤에 ?v=33 붙여서 접속
  2. Ctrl + Shift + R 강제 새로고침
  3. 시크릿 창(Ctrl+Shift+N)에서 접속
  4. 여전히 안 되면 관리자에게 문의

우측 하단의 버전 표시(예: v3.3)로 현재 버전을 확인할 수 있습니다.
