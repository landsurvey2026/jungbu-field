Section 01

시스템 한눈에 보기

이 시스템은 중부수도사업소 직원 5~8명이 함께 사용하는 공사지시 관리 도구입니다. 모두가 같은 화면을 보고, 한 명이 입력하면 다른 사람 핸드폰에 즉시 반영됩니다. 카카오톡으로 정보를 전달하다가 누락되는 일이 없어집니다.

이 프로그램으로 할 수 있는 일

  • ✓공사지시 등록 — 지시번호·공사명·소장 정보·연장 등 기본정보 입력
  • ✓진행 단계 추적 — 공사예정 → 공사중 → 공사완료 → 측량완료 → 로딩완료
  • ✓준공도 인수 관리 — 인수일부터 14일 안에 로딩 완료해야 함을 자동 감시
  • ✓자동 알림 — 인수일 7일·10일·13일 경과 시 주의/경고/위험 알림
  • ✓검색·필터 — 지시번호·공사명·위치·소장명으로 즉시 검색
  • ✓채팅·메모 — 각 지시서별로 대화 기록을 자동 보존
  • ✓엑셀 내보내기 — 관리자가 PC에서 정리된 양식으로 다운로드

두 가지 역할

역할설명
일반 사용자
최대 8명		 현장 직원. 핸드폰·PC 모두 사용 가능. 공사지시 등록·조회·진행 단계 변경 가능. 관리자가 미리 이름을 등록해야 함.
관리자
PC 최대 3대		 사업소 책임자 (보통 1~2명). PC에서만 접속 가능. 사용자 등록, 비밀번호 변경, 엑셀 내보내기 등 추가 기능 사용.
💡 핵심 원칙 카카오톡·전화로 알리던 것을 시스템 안에서 처리합니다. "그때 누구한테 말했었지?" 같은 일이 사라집니다.

준공도 인수 처리

로딩 마감 섹션에 D+0, D+1, ... 표시됨

자동 알림 일정

시점알림설명
인수일 + 7일 🟡 주의 여유 7일. 슬슬 로딩 준비 시작
인수일 + 10일 🟠 경고 여유 4일. 빨리 처리해야 함
인수일 + 13일 🔴 위험 내일 마감. 상단 빨간 배너 + 깜빡임
인수일 + 14일 규정상 로딩 완료 마감일
🎯 알림 자동 종료 "로딩완료" 단계가 체크되면 알림이 즉시 멈춥니다. 이미 처리한 건에 대해 계속 알림 받지 않습니다.

로딩 완료 후 — 최종연장 입력

로딩완료 단계로 진행하면 최종연장 입력칸이 나타납니다. 실제 시공한 연장을 입력하세요.

Section 08

검색·필터·보기 모드

공사지시가 많아지면 원하는 건을 빠르게 찾는 기능이 필요합니다.

검색

화면 상단의 검색창에 입력하면 실시간으로 필터링됩니다. 다음 항목 모두 동시에 검색됩니다:

  • ✓지시번호 (예: 001 입력 시 중부001 매칭)
  • ✓공사명 (예: 창신)
  • ✓공사위치
  • ✓소장명 (예: 홍길동)
  • ✓전화번호

탭으로 단계별 필터링

상단의 7개 탭을 누르면 해당 단계의 건만 보입니다.

보기 모드 3가지

모드화면언제 쓰나

📜 간단 한 줄에 지시번호+공사명+상태 공사건이 많을 때 한눈에 보기

관리자 패널 (PC 전용)

관리자로 로그인하면 우측 상단에 ⚙️ 관리자 버튼이 나타납니다. 이 버튼을 누르면 관리자 패널이 열립니다.

⚠ PC 전용 관리자는 보안상 PC에서만 접속 가능합니다. 핸드폰에서 관리자 비밀번호를 입력해도 차단됩니다. PC는 최대 3대까지 등록 가능합니다.

관리자 패널 — 6개 카드

카드기능
🔗 공유 주소 직원들에게 공유할 접속 주소를 복사·저장
👥 등록된 사용자 최대 8명까지 직원 등록·제거. 마지막 접속일 확인
🔑 사용자 비밀번호 일반 직원들이 사용할 비밀번호 변경
🏷️ 지시번호 접두어 "중부", "강동" 등 사업소 이름 변경 (전체 화면에 자동 반영)
🔐 관리자 비밀번호 관리자 본인 접속용 비밀번호 변경
💻 등록된 관리자 PC 최대 3대 등록·제거

사용자 등록 절차

관리자로 로그인 → ⚙️ 관리자 클릭
👥 등록된 사용자 섹션 찾기
새 사용자 이름 입력 (예: 김철수) → ➕ 추가
최대 8명까지 반복
직원들에게 "이 비밀번호로 본인 이름으로 접속하세요" 안내
🚨 이름 정확히 등록 직원이 평소 쓰는 이름과 정확히 같게 등록해야 합니다. 띄어쓰기·한자/한글 등 사소한 차이도 인식 못 합니다.
잘못 등록 시 → 직원이 "🚫 등록되지 않은 사용자입니다" 에러 받음.

자주 묻는 질문

로그인이 안 돼요
"등록되지 않은 사용자입니다" 메시지가 뜨면 관리자에게 본인 이름 등록 요청하세요. "비밀번호가 틀렸습니다"면 관리자에게 비밀번호 확인 요청.
알림이 안 와요
① 홈 화면 아이콘으로 실행했는지 확인 (사파리·크롬 탭 ❌) ② 첫 접속 시 알림 권한 "허용" 눌렀는지 확인 ③ 핸드폰 설정 → 앱 알림에서 차단 안 됐는지 확인 ④ 아이폰은 iOS 16.4 이상 필요.
데이터가 안 보이거나 이상해요
화면을 새로고침(아래로 당기기) 하거나, PWA 앱을 완전히 종료하고 다시 실행해보세요. 그래도 안 되면 관리자에게 문의.
지시번호를 잘못 입력했어요
카드를 누르고 상세화면에서 "기본정보 수정" 클릭 → 수정. 단, 중복되면 저장이 안 됩니다 (미지시 제외).
실수로 다음 단계를 눌렀어요
이전 단계를 누르면 그 단계로 되돌아갑니다. 예: "공사완료"에서 실수로 "측량완료"를 누른 경우 → "공사완료"를 다시 누르면 복구.
"미지시"는 왜 중복 허용되나요?
정식 지시번호가 안 나온 공사를 여러 건 동시에 등록할 수 있게 하기 위함입니다. 정식 번호가 나오면 수정해서 변경하세요.
소장 이름이 자동완성 안 돼요
처음 등록하는 소장이라 시스템에 학습된 데이터가 없어서 그렇습니다. 한 번 등록하면 다음부터는 자동 추천됩니다. 같은 이름 다시 선택하면 전화번호도 자동 채워집니다.
엑셀 버튼이 안 보여요
엑셀 내보내기는 PC에서만 가능합니다. 핸드폰에서는 버튼이 숨겨져 있습니다. 관리자에게 요청하세요.
관리자 비밀번호를 잊었어요
또 다른 관리자 PC에서 로그인 후 변경하거나, 시스템 제작자에게 문의하세요. (재설정은 HTML 파일과 Firebase 데이터 수정 필요)
사업소를 다른 곳으로 옮겨도 쓸 수 있나요?
네. 관리자 패널 → 지시번호 접두어 → "강동" 등으로 변경하면 사업소 이름과 자동 접두어가 모두 같이 바뀝니다.
데이터가 사라질 수 있나요?
데이터는 Google Firebase에 자동 백업됩니다. 다만 관리자 패널에서 "사용자 제거"나 "지시서 삭제" 시에는 영구 삭제되니 신중히.
8명보다 더 많이 등록하고 싶어요
현재 코드 상 8명으로 제한. 시스템 제작자에게 수정 요청하면 됩니다. (HTML 파일 한 줄 수정으로 가능)
